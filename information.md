# 시작 명령어

`npm i`
`npm run server`
`npm start`
`npm install --save-dev cra-bundle-analyzer` // 번들 파일 분석을 위해 설치
`npx cra-bundle-analyzer` // 번들 파일 확인 명령어
`npm run serve`
